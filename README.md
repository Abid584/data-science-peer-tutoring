# 📊 Data Science Peer Tutoring

Welcome to the **Data Science Peer Tutoring** repository! This is your one-stop resource for lecture notes, Python notebooks, and learning materials covered in our tutoring sessions.

---

## 📚 Topics Covered

### 🐍 Python Fundamentals
- Variables, data types, and operators
- Control flow (if/else, loops)
- Functions and modules
- Lists, tuples, dictionaries, and sets

---

### 🐼 Pandas
- DataFrames and Series
- Reading & writing data (CSV, Excel)
- Data cleaning and handling missing values
- Filtering, grouping, and aggregation
- Merging and joining DataFrames

---

### 🔢 NumPy
- **1D Arrays** — array creation, indexing, slicing, and operations
- **2D Arrays** — matrix operations, reshaping, broadcasting, and linear algebra basics

---

### 📈 Matplotlib
- Line plots, bar charts, histograms, and scatter plots
- Customizing titles, labels, and legends
- Subplots and figure sizing

---

### 🎨 Seaborn
- Statistical visualizations (heatmaps, boxplots, violin plots)
- Distribution and relationship plots
- Styling and themes

---

### 📐 Statistics for Data Science
- Measures of central tendency (mean, median, mode)
- Measures of spread (variance, standard deviation, range)
- Probability basics
- Distributions (normal, binomial)
- Correlation and covariance
- Hypothesis testing basics

---

### 🤖 Basics of Machine Learning
- What is Machine Learning?
- Types of ML: Supervised, Unsupervised, Reinforcement Learning
- ML workflow: data preparation → training → evaluation
- Train/test split and model evaluation metrics

---

### 🔍 K-Nearest Neighbors (KNN)
- How the KNN algorithm works
- Choosing the right value of K
- KNN for classification and regression
- Implementation using Scikit-learn

---

## 🚀 How to Use

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/data-science-peer-tutoring.git
   ```

2. **Navigate to any topic folder** and open the `.ipynb` notebook in Jupyter or Google Colab.

3. **Run on Google Colab** *(no installation needed)*  
   Open any notebook directly in your browser by clicking the Colab badge inside each notebook.

## 🛠️ Requirements

We will be using **Anaconda** for this course — it comes with all the required libraries pre-installed (NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn, Jupyter Notebook and more).

- 📥 Download Anaconda: [https://www.anaconda.com/download](https://www.anaconda.com/download)
- After installation, simply open **Jupyter Notebook** from Anaconda Navigator — no extra setup needed!
